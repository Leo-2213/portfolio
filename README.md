# Abhijeet Atmapoojya — Personal Portfolio

A fully responsive, modern personal portfolio website built with React, Tailwind CSS, and custom animations.

## 🚀 Quick Start

No build step needed! Just open `index.html` in any browser.

## 🌐 Deploy in 30 seconds

### Option 1 — Netlify Drop (Easiest)
1. Go to https://app.netlify.com/drop
2. Drag and drop the `index.html` file
3. Share your live link instantly

### Option 2 — GitHub Pages
1. Create a new GitHub repo
2. Upload `index.html` (rename to `index.html` if needed)
3. Go to Settings → Pages → Branch: main → Save
4. Live at: https://yourusername.github.io/repo-name

### Option 3 — Vercel
1. Go to https://vercel.com
2. Drag and drop the folder
3. Get a live link instantly

## ✏️ Customization

All your personal data is in the `DATA` object near the top of the `<script>` tag in `index.html`.

Update:
- `DATA.email` — your email
- `DATA.linkedin` — your LinkedIn URL
- `DATA.github` — your GitHub URL
- `DATA.projects` — add more projects
- `DATA.skills` — adjust skill levels

## 🛠 Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Custom CSS animations
- Canvas API (particle background)
- Intersection Observer API (scroll animations)

## 📦 Features

- Dark / Light theme toggle
- Typing effect in hero
- Particle canvas background
- Scroll-triggered animations
- Fully responsive (mobile + desktop)
- Sticky navbar with active section highlight
- Cursor glow effect
- Contact form UI
- Glassmorphism cards
